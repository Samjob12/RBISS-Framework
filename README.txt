RBISS Framework Reproducibility Guide

Framework:
Rule-Based Indicator Selection Score (RBISS)

Description:
This framework provides a scale-flexible and normalization-based qualitative screening procedure for transparent and reproducible indicator selection.

Files Included:
- RBISS_framework.py
- requirements.txt
- RBISS_framework_scale_standardization_table.png
- RBISS_framework_scale_standardization_table.pdf

Procedure to Run:

1. Install Python 3.11 or above

2. Install required libraries:

pip install -r requirements.txt

3. Run the framework:

python RBISS_framework.py

4. The framework automatically generates the RBISS visualization figure.
